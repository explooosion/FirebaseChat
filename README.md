# FirebaseChat
firebase chat build with angular2

## Install

```bash
git  clone https://github.com/explooosion/FirebaseChat.git

cd FirebaseChat

npm install
```

## Run
```bash
ng serve
```

## Export Dist
```bash
ng build
```
